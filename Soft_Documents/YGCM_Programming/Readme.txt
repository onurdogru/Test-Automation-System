Follow the steps mentioned in this file to program the device:

1. Connect MiniProg3 to the target device and the laptop. Ensure the programming conditions are met.
2. Open the HEX-file folder. Delete the hex file in the folder.
3. Copy the hex file that you want to be programmed into the HEX-file folder.
4. Run the PSoC4.bat script.
5. If in case this does not work, run UpdateDLL.bat and then run PSoC4.bat script.
6. Please let me know your results.
